Privacy Policy

Last updated: March 27, 2026

Harness ("the Plugin") is an open-source Claude Code plugin distributed under the Apache 2.0 license. This Privacy Policy explains how Harness handles user data.

The short version

Harness does not collect, store, transmit, or share any personal data. It runs entirely within your local Claude Code environment.

1. Data Collection

Harness does not collect any personal information. Specifically:

  • No user accounts or registration required
  • No analytics, telemetry, or tracking of any kind
  • No cookies or local storage for tracking purposes
  • No network requests to external servers
  • No collection of IP addresses, device identifiers, or usage patterns

2. How the Plugin Works

Harness operates exclusively within the Claude Code runtime environment:

  • It reads your project files to analyze the domain and codebase
  • It generates Markdown files (.claude/agents/ and .claude/skills/) written to your local project directory
  • All processing happens locally through Claude Code's built-in tools
  • No data leaves your machine through the plugin itself

3. Third-Party Services

Harness does not integrate with or send data to any third-party services. However, please note:

  • Harness runs within Claude Code, which is operated by Anthropic. Your interactions with Claude Code are subject to Anthropic's Privacy Policy.
  • If you install Harness via a GitHub marketplace, your interaction with GitHub is subject to GitHub's Privacy Statement.

4. Generated Files

Files generated by Harness (agent definitions, skills, orchestrator configurations) are stored locally in your project directory. You have full control over these files — they can be modified, deleted, or version-controlled as you see fit.

5. Open Source Transparency

Harness is fully open source. You can audit the complete source code at github.com/tobyilee/dynamic-harness to verify these privacy claims.

6. Children's Privacy

Harness does not knowingly collect any data from anyone, including children under 13. Since no data is collected at all, there is no age-specific concern.

7. Changes to This Policy

If this privacy policy changes, the updated version will be published in the GitHub repository and on this page. The "Last updated" date at the top will reflect the most recent revision.

8. Contact

For questions about this privacy policy or the Harness plugin, please open an issue at github.com/tobyilee/dynamic-harness/issues or contact the maintainer via GitHub.
